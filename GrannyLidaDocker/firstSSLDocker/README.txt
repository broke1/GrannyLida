Это папка служит для первичного получения сертификатов

Запусти команду ./run_certbot.sh

  Он может ругаться, что Already running certbot или как-то так. Просто еще раз запусти.

  Он будет предлагать выбор либо 1. Создать новый. 2. Перевыпустить сертификат. 
  Я выбирал 2.
  
  В итоге в папке выше (родительской) появиться папка cert с сертификатами

  Дальше нужно скопировать папку live из certs в папку certbot/conf
